## Migration Guide

If you encounter breaking changes from v1.0.0, follow these steps to update your integration seamlessly.

<Steps>
  <Step title="Update SDK" icon="download">

Install the latest Kohhee Peace SDK.

<CodeGroup tabs="npm,yarn">
````bash
npm install @kohheepeace/sdk@latest
````
````bash
yarn add @kohheepeace/sdk@latest
````
</CodeGroup>

  </Step>
  <Step title="Replace API Keys" icon="key">

Replace legacy API keys with JWT tokens.

````javascript
// Old
const client = new KohheeClient({ apiKey: 'YOUR_API_KEY' });

// New
const token = await getJWTToken();
const client = new KohheeClient({ token });
````

  </Step>
  <Step title="Test Endpoints" icon="play">

Verify your calls to `https://api.example.com/v1/repos`.

<Callout kind="tip">
Run `npm test` to validate migrations.
</Callout>

  </Step>
</Steps>
