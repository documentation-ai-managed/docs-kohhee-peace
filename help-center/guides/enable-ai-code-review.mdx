---
title: Enable AI-Powered Code Review
description: Turn on AI-assisted code reviews in your Kohhee Peace repository.
---

## Introduction
AI code review provides suggestions and detects issues on pull requests.

## Steps
<Steps>
  <Step title="Go to your repository settings">
    ````
    Navigate to the repository you want AI reviews on in Kohhee Peace.
    ````
  </Step>
  <Step title="Enable AI review">
    ````
    Under **Integrations**, toggle **AI-Powered Code Review** to ON.
    ````
  </Step>
  <Step title="Open a pull request">
    ````
    Submit a PR. The AI assistant will comment automatically with suggestions.
    ````
  </Step>
</Steps>
